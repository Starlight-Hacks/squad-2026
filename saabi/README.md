# SAABI - The Engine of the Informal Economy

SAABI is a revolutionary fintech ecosystem designed to bridge the gap between Nigeria's informal traders and formal financial institutions. Built for the **Squad Hackathon**, SAABI leverages natural language processing on WhatsApp and state-of-the-art data visualization to create a seamless marketplace and research network.

## Key Features

- **3D Interactive Experience**: Immersive digital globe representing West Africa's economic connectivity.
- **Distributed Payments**: Zero-install payment initiation via WhatsApp using natural language ("Send 2500 naira to...").
- **Service Discovery**: AI-powered matching between informal workers and potential clients.
- **State-wide Dashboard**: Institutional visibility into LGA-level transaction volumes, heatmaps, and sector health.
- **Privacy Centric**: Aggregative data modeling with a minimum cohort threshold of five users for all public reporting.

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS 4.0
- **3D Graphics**: Three.js
- **Animations**: GSAP (GreenSock), Framer Motion
- **Visualizations**: Chart.js, React-Chartjs-2
- **Icons**: Lucide React

## Installation

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```
